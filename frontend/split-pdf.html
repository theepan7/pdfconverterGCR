    How to Split PDFs Online

    1. Drop your PDF files onto the upload area, or click to browse.
    2. Click Add more files to pick PDFs stored in other folders.
    3. Set a Start and End page for each file you want to extract from.
    4. Drag the rows up or down to set the processing order.
    5. Click Split PDFs — each extracted range downloads as a separate file.

    When to Split a PDF

    Splitting a PDF is handy when you need to share only part of a document, extract a chapter from a book, or separate invoices bundled into a single file.
