What is Mini PDF Tools?

Mini PDF Tools is a free online PDF toolkit that lets you compress, merge, split, rotate, delete pages, convert images to PDF, and export PDF pages as images — all without creating an account or installing anything.

We built it because most PDF tools online either have aggressive paywalls, show intrusive ads, or make you hand over an email address just to use basic features. We think that's unnecessary. PDF manipulation is a simple technical task — it should be free and private by default.

Every tool on this site works entirely through your browser and our secure cloud processing API. Your files are processed in memory, never written to disk, and are never stored on our servers.

What we stand for

Privacy first

Your files are never stored, logged, or shared. Processing happens in memory and the data is discarded the moment your download starts.

Actually free

No file size limits hidden behind a paywall. No "free trial" that expires. No account required. Every tool is free for everyone, every time.

Fast by design

Powered by Google Cloud Run — processing starts in milliseconds and scales automatically. No queuing, no waiting for a server to wake up.

Works everywhere

No software to download. Works on any device with a browser — desktop, tablet, or mobile — on Chrome, Safari, Firefox, and Edge.

How it works

When you upload a file and click a tool button, your file is sent over an encrypted HTTPS connection to our processing API hosted on Google Cloud Run. The API performs the requested operation — compressing, merging, splitting, rotating, or converting — entirely in server memory.

The result is sent back to your browser as a download. At no point is your file written to a database, cloud storage bucket, or any persistent disk. The moment the response is sent, the data is gone.

All connections use TLS 1.2 or higher. We never transmit your files to any third party.
